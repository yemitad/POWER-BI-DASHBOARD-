# Power BI HR Analytics Dashboard

This project is an interactive **HR Analytics Dashboard** built in **Power BI** to help analyze workforce trends, employee demographics, and attrition patterns.

## Project Overview
The dashboard provides insights into:  
- Total employee count
- Attrition rate
- Employee demographics
- Department and job role analysis
- Salary and compensation trends
- Job satisfaction and work-life balance

## Features
- Interactive slicers for filtering data
- KPI cards for high-level HR metrics
- Bar charts, pie charts, and line charts for trend analysis
- Attrition analysis by department, role, age group, and gender
- Employee distribution by education field and marital status
- Drill-down capabilities for deeper analysis

## Tools Used
- Power BI
- Power Query
- DAX

## Dashboard Insights
This dashboard helps HR teams:
- identify departments with high attrition
- understand employee demographic patterns
- monitor compensation and performance indicators
- support data-driven workforce decisions
